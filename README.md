zhihu_download，爬取某博主在知乎上的所有文章。要注意cookie的获取方式：
  - 网页打开想下载的博主的主页，如果想下载他的回答，要点击相应的“回答”
    <img width="1368" height="716" alt="image" src="https://github.com/user-attachments/assets/2bde688b-3e7b-4ce1-8c86-efea06a4eaa2" />
  - 然后F12，点击右侧的network，可以看到图中的cookie区域，<img width="1394" height="976" alt="image" src="https://github.com/user-attachments/assets/0062ce26-1f04-4385-9b3c-eab5e9001d27" />
  - 如果是想下载“文章”,则点击相应的选项卡，再重新获取cookie即可。
